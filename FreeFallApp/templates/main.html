Для чего служит этот сайт?

Проект разрабатывается для сбора туристами группы для спортивных походов. Каждый пользователь может предложить маршрут и начать сбор группы по нему, или же добавиться в группу другого человека. Этот сервис может помочь, если, например, пользователь только хочет начать ходить в походы, или же уже опытный турист собирает компаньонов из разных городов, однако облась применения сервиса несколько шире, и он может использоваться просто для удобства сбора группы внутри клуба (в таких случаях, просим в кратком описании делать соответствующие уточнения).
